## Run Locally

Clone the project

```bash
  git clone https://github.com/Rohit-14361/star-rating.git
```

Go to the project directory

```bash
  cd star-rating
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
